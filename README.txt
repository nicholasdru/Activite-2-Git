Bonjour, ce read me est un fichier avec des commande et notions de base pour utiliser le terminat distribué Git.

PREMIEREMENT - Vous pouvez entrez help pour voir une liste de commandes utliser dans se terminal.
	     - Ensuite vous pouvez aussi regarder en ligne pour trouvez autre commandes.


Apres l'instalation veuiller choisir un 'repository' pour votre fichier (Principal (.Git))
Vous devez le metre dans doccument ou autre endroit, pas dans votre C://user/'nomUser'..

si vous lavez instaler dans 'user', votre 'Terminal' prendra compte de beaucoup trop de fichier inutile...

pour ignorer ces fichier veuiller entrez la commande "git .ignore" et avec celle ci vous pouvez entrez
le nom des fichier qui sont listé dans votre "Git status"! Ceci peu s'averez compliquer mais vous pouvez
 toujour me contacter pour avoir de l'aide.
--------------------------------------------------------------

Pour debuter il faudrait savoir qu'ece qu'un commit
>-chaque changement au formulaire / version du code pour ne pas perde

Un message descriptif avec raisson d'etre devrai etre entrez avec chaque commit
Pratiquement entrez 'git commit' avec '-a' pour enregistrez tous les fichier local qui on eu changement.
				 avec '-am' pour enregistrez tous les fichier et message ""
